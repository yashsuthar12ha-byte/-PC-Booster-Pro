## ✨ Features

| # | Optimization | What it does |
|---|-------------|-------------|
| 🛑 | Kill Bloat | Terminates OneDrive, Teams, Spotify, Xbox, Edge Updater & more |
| ⚙️ | Stop Services | Disables 10 unnecessary Windows services (Telemetry, SysMain, etc.) |
| 🎮 | Game Mode | Enables Game Mode, disables animations & transparency |
| 🗑️ | Clean Junk | Removes Temp, Prefetch & cache files |
| 🧠 | RAM Trim | Empties working set on every running process |
| ⚡ | Power Plan | Activates High Performance / Ultimate power plan |
| 🚀 | Startup Clean | Removes bloat from registry startup entries |
| 🌐 | Network Reset | Flushes DNS & resets Winsock |

---

## 🚀 How to Use

1. **Download** the `.exe` file
2. **Right-click** → **Run as Administrator** (required for full optimization)
3. Click **BOOST MY PC**
4. Wait 10-15 seconds
5. **Restart your PC** for full effect

## 🔧 Build from Source

Requirements: Windows with .NET Framework 4.8 (built-in on Windows 10/11)

```cmd
csc /target:winexe /reference:System.Windows.Forms.dll /reference:System.Management.dll /reference:System.ServiceProcess.dll /out:PC_Booster_Pro_v2.0.exe src\PC_Booster_Pro.cs
```
